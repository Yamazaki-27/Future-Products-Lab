# Sample Prompt

## publish-report 実行例

```text
publish-report.md のスキルを使って、現在の Report.md をGitHub公開前の品質で最終確認してください。画像リンク、Markdown構文、README.md、CHANGELOG.md、Release Notes、PUBLISH_SUMMARY.md、公開判定まで実施してください。必要な修正は確認なしで行ってください。
```

## review + publish 実行例

```text
review-report.md で Report.md の写真配置と本文の流れを編集者の視点で改善し、その後 publish-report.md でGitHub公開前チェックを行ってください。
```
